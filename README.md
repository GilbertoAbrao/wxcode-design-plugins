# wxcode-design-plugins

WXCode-curated Open Design plugin marketplace.

Catalog of first-party admin / dashboard / cockpit plugins ready to install
on any Open Design daemon. Tenants register this repo's `registry/wxcode-marketplace.json`
as a marketplace source and pull plugins from `plugins/<slug>/` on demand.

## Layout

```
plugins/
  admin-neobrutal/         # plugin source (open-design.json, SKILL.md, example.html, ...)
  admin-finance-terminal/
  admin-saas-bestseller/
  admin-glassmorphism/
  admin-ai-copilot/
registry/
  wxcode-marketplace.json  # marketplace.v1 catalog — pointed at by daemon sources
```

## Install a single plugin (any Open Design daemon)

```bash
od plugin install github:GilbertoAbrao/wxcode-design-plugins@main/plugins/admin-neobrutal
```

## Register the whole catalog (UI: Plugins → Sources → Add source)

```
https://raw.githubusercontent.com/GilbertoAbrao/wxcode-design-plugins/main/registry/wxcode-marketplace.json
```

All entries then appear under **Plugins → Available** with 1-click install.

## Catalog spec

`registry/wxcode-marketplace.json` follows the
[Open Design marketplace.v1 schema](https://open-design.ai/schemas/marketplace.v1.json).
See `plugins/<slug>/open-design.json` for individual plugin manifests.

## Authoring

Plugins here are generated by the `/od-plugin-forge` Claude Code skill
(marketplace mode). Each plugin is a single-file HTML template + a SKILL.md
authoring brief + a manifest. See `plugins/admin-neobrutal/` for the canonical
shape.
